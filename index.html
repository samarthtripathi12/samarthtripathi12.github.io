PROJECT 01

Black Hole Light Bending

Computational General Relativity

Derived geodesic equations from the Schwarzschild metric and numerically integrated photon trajectories around a black hole. Verifies the photon sphere at r = 1.5Rₛ with sub-10⁻³ relative error using RK4 integration. Governing equations independently derived from first principles, July 2023.

PROJECT 02

Gravitational Wave Denoising

LIGO Real Data · Signal Processing

Computational analysis pipeline applied to real LIGO H1 strain data (GW150914). Covers FFT spectral analysis, 4th-order Butterworth band-pass filtering at 50–600 Hz, and detection significance via SNR proxy. Fourier equations derived from first principles, August 2023.

PROJECT 03

Neural Network From Scratch

Deep Learning · Algorithmic Foundations

Fully functional neural network built using only NumPy. Covers forward propagation, backpropagation via chain rule, and gradient descent weight updates. Achieves 99.5% accuracy (199/200) on XOR dataset. All equations independently derived. Dated October 2023.
