01 — Computational Physics

Computational Simulation of Light Trajectories Near Black Holes

A progressive study from Newtonian approximation to Schwarzschild GR and Kerr spacetime. Governing equations derived independently from first principles beginning July 2023. Quantitatively verifies the photon sphere at r = 1.5Rs with sub-10⁻³ relative error using RK4 integration. Dated notebook derivations included as documentary evidence.

02 — Signal Processing

Gravitational Wave Signal Extraction from Real LIGO Data

Computational analysis pipeline applied to real LIGO H1 strain data (GW150914). Framework covers raw data visualisation, FFT spectral analysis, 4th-order Butterworth band-pass filtering at 50–600 Hz, and detection significance estimation. Fourier transform and SNR proxy equations derived from first principles, August 2023.

03 — Machine Learning

Neural Network from Scratch: XOR Classification

Fully functional neural network built using only NumPy. Covers forward propagation, backpropagation via chain rule, gradient descent weight updates, and training dynamics visualisation. Achieves 99.5% accuracy (199/200) on the XOR dataset. All equations derived independently. Includes critical analysis of generalisation limits under real-world stochastic noise conditions. Dated October 2023.
