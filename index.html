PROJECT 01

Black Hole Light Bending

Computational General Relativity

Derived the geodesic equations from the Schwarzschild metric and numerically integrated photon trajectories around a black hole, visualizing the photon sphere and gravitational lensing effects. The simulation produces accurate bending angle predictions consistent with analytical GR results, built entirely from scratch in Python. This project demonstrates a rigorous command of tensor calculus, numerical ODE solving, and relativistic physics.

General Relativity Numerical ODE Python Geodesics

PROJECT 02

Gravitational Wave Denoising

LIGO Real Data · Signal Processing

Applied Fourier transforms, bandpass filtering, and wavelet decomposition to raw LIGO strain data to isolate gravitational wave signals from detector noise. The pipeline reproduces the GW150914 chirp signature and evaluates denoising performance through SNR analysis. This work required deep engagement with the physics of compact binary mergers and the mathematics of spectral and time-frequency analysis on real scientific data.

Signal Processing LIGO Data Wavelet Analysis FFT

PROJECT 03

Neural Network From Scratch

Deep Learning · Algorithmic Foundations

Implemented a fully-connected neural network from the ground up in NumPy — deriving backpropagation using the chain rule, coding gradient descent with momentum, and validating on benchmark classification tasks. No ML frameworks were used; every matrix multiply, activation, and weight update was written explicitly. The project demonstrates a genuine understanding of deep learning mechanics that goes well beyond using existing libraries.

Backpropagation Gradient Descent NumPy Classification
